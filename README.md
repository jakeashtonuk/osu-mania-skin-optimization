# osu!mania Skin Optimization

## Download

[Download IceTea-UK-Ultimate-Performance-Edit.osk](https://github.com/jakeashtonuk/osu-mania-skin-optimization/releases/latest/download/IceTea-UK-Ultimate-Performance-Edit.osk)

## Install

1. Download the `.osk` file from the link above.
2. Double-click the downloaded `.osk` file.
3. osu! will import the skin automatically.

## Credit and provenance

This repository does not contain a wholly original skin created by me.

It is an enhanced, optimised, and performance-focused edit of an existing osu!mania skin lineage that traces back to prank855’s pl0x skin, later Myuka / IceTea-related edits, a publicly circulated Eliminate294 variant, and a later JOSHUA0309 remix. Publicly available sources show prank855’s original pl0x release in 2017, a Myuka-associated red-arrow derivative by January 2022, an Eliminate294 circles variant by February 2022, and a JOSHUA0309 remix thread in January 2024 that explicitly described the skin as a combination of borrowed elements rather than an original work.

My contribution in this repository is limited to enhancement, optimisation, configuration changes, usability refinements, and performance-oriented adjustments. I am not claiming original ownership of the base skin or of all included assets.
